# 🚀 JobFit AI 
<<<<<<< HEAD

AI-powered Resume Optimization & ATS Analysis Platform built using FastAPI, React.js, NLP, and AI APIs.

JobFit AI helps users:
- Analyze resumes
- Calculate ATS score
- Match resumes with job descriptions
- Detect missing skills
- Generate AI-optimized ATS resumes
- Download professional recruiter-ready resumes

---

# 🌟 Features

## ✅ ATS Resume Analysis
- Extract resume text from PDF
- Detect technical skills
- Analyze ATS compatibility
- Provide resume improvement feedback

## ✅ Job Description Matching
- Compare resume with job description
- Calculate job match percentage
- Identify matching skills
- Improve resume according to role requirements

## ✅ AI Resume Optimization
- AI-powered resume rewriting
- ATS-friendly optimization
- Improved professional summaries
- Enhanced project descriptions
- Better keyword optimization

## ✅ Professional Resume Generation
- One-page recruiter-style resumes
- Dynamic PDF generation
- Clean ATS formatting
- Download optimized resume instantly

## ✅ Dynamic Resume Parsing
Automatically extracts:
- Name
- Email
- Phone Number
- LinkedIn
- GitHub

## ✅ Modern Frontend UI
- Responsive dashboard
- Glassmorphism UI
- Interactive analytics
- Professional recruiter-style design

⚙️ Installation
1️⃣ Clone Repository
git clone https://github.com/yourusername/JobFit-AI.git

cd JobFit-AI
🚀 Backend Setup
Create Virtual Environment
python -m venv venv
Activate Environment
Windows
venv\Scripts\activate
Mac/Linux
source venv/bin/activate
Install Dependencies
pip install -r requirements.txt
🔐 Environment Variables

Create .env inside backend/

OPENROUTER_API_KEY=your_api_key_here
▶️ Run Backend
uvicorn main:app --reload

Backend runs on:

http://127.0.0.1:8000
🎨 Frontend Setup
cd frontend
Install Packages
npm install
Run Frontend
npm run dev

Frontend runs on:

http://localhost:5173
